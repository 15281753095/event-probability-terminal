# Binance Wallet Prediction Markets Notes

Verification date: 2026-04-21 Asia/Shanghai.

Source:

- https://www.binance.com/en/academy/articles/a-guide-to-binance-wallet-prediction-markets

## Verified facts

- Binance Academy describes Binance Wallet Prediction Markets as an integration of the third-party Predict.fun decentralized prediction market protocol on BNB Smart Chain.
- Binance Academy describes Binance as an access layer that aggregates and displays these markets in the Binance app.
- Binance Academy says Binance does not create events and does not act as counterparty to prediction trades.
- Binance Academy describes product semantics including YES/NO shares, market/limit orders, market phases, Prediction Account, gas sponsorship, and keyless wallet infrastructure.
- Binance Academy notes availability may vary by region.

## Project decisions

- Binance Wallet Prediction Markets are product-semantics reference only in Phase 1.
- No Binance Wallet adapter is allowed until an official developer API reference is identified and documented.
- No Binance account, wallet, funding, keyless-wallet, transfer, settlement, or trading integration is allowed.

## Reasonable inferences

- Binance Wallet can inform how the terminal labels phases and YES/NO outcome semantics.
- Predict.fun should be treated as the protocol/API research target behind the Binance Wallet reference surface.
